## Summary

Gather AI builds a vision-powered platform using autonomous drones to capture real-time warehouse data, digitizing supply chain workflows. This is their **first Growth Engineer** (listed as "GTM Engineer" on LinkedIn), sitting on the Marketing team with a dotted line to Engineering. You own the GTM data stack, build AI-powered internal tools (lead scoring, sales summaries, churn prediction), create prospect-facing experiences (interactive demos, ROI calculators), and run A/B experiments. **Code directly impacts revenue** — full-stack building, not app-connecting.

## Responsibilities

- Build AI-powered GTM tools using company data and AI capabilities: AI-assisted lead scoring, automated sales summaries, intelligent churn prediction
- Create high-impact customer journey experiences: self-serve interactive product demos, ROI calculators, prospect-facing web applications
- Own and instrument the GTM data stack: technical ownership for data flow between product, website, and GTM systems (Salesforce, HubSpot)
- Run experiments: design, build, and analyze A/B tests on website, email, and other platforms to optimize conversion rates and drive growth

## Requirements

### Must-Have
- AI-native builder — hands-on experience building with LLMs, AI APIs (like OpenAI's), or other ML frameworks
- Full-stack engineer — Python and SQL for data pipelines, JavaScript (React) for front-end
- Product sense — proven ability to understand user needs and build lovable tools
- Data-driven and curious — dig into data, form hypotheses, build experiments
- Scrappy and resourceful self-starter in a fast-paced startup

### Nice-to-Have
- None explicitly stated

## Key Concepts

- **Interactive Product Demo**: A self-serve, web-based experience that lets prospects explore the product without a sales call — increasingly used in PLG and technical-buyer sales motions to accelerate pipeline by letting buyers experience value before committing to a meeting.
- **GTM Data Stack**: The set of systems and pipelines that move data between a company's product, website, and revenue tools (CRM, marketing automation). This role owns the code that keeps that data clean, connected, and actionable — not just the tool configuration.
