## Summary

Tempo Software is the #1 time management add-on for Jira with 30,000+ customers including a third of Fortune 500 companies, offering a suite of integrated solutions for time management, resource planning, budget management, and roadmapping in the Atlassian ecosystem. This GTM Engineer role architects the revenue systems powering Tempo's next growth phase — building autonomous AI SDRs, RAG applications with vector databases, composable GTM architecture with n8n as the central orchestration platform, and sophisticated lead scoring/enrichment pipelines with Clay. One of the most technically demanding GTM Engineer roles in the market.

## Responsibilities

### AI-Powered Revenue Generation
- Build autonomous AI SDRs that research prospects, craft personalized messages, and manage follow-up sequences at scale
- Develop sophisticated lead scoring and enrichment pipelines using Clay data providers and waterfall enrichment
- Create AI-powered personalization engines using LLMs for dynamic content generation
- Design intent-based trigger systems that identify and act on buying signals automatically
- Implement in-app and chat automations for user journey guidance and expansion opportunities

### Data Pipeline & Intelligence Systems
- Develop RAG applications and vector databases (Supabase) for intelligent content retrieval and account intelligence
- Create real-time dashboards tracking pipeline velocity, conversion rates, and revenue efficiency
- Implement predictive models for churn prevention and expansion opportunity identification
- Design automated lead routing logic considering rep expertise, capacity, and historical performance
- Create customer lifecycle triggers for upsell opportunities and churn prevention

### System Integration & Automation Architecture
- Design composable GTM architecture using n8n as central orchestration platform
- Build event-driven workflows responding to buyer signals in real-time
- Create reusable automation patterns and maintain APIs

## Key Concepts

- **RAG (Retrieval-Augmented Generation)**: AI architecture pattern that retrieves relevant documents from a vector database before generating responses — used here to power intelligent account intelligence and content retrieval for sales conversations.
- **Waterfall Enrichment**: Sequential data enrichment strategy that queries multiple data providers in priority order, falling through to the next source when the previous one returns no match — maximizes coverage while controlling cost.
- **Composable GTM Architecture**: Modular system design where individual automation components (enrichment, scoring, routing, outreach) can be independently built, tested, and recombined — with n8n as the central orchestration layer connecting all pieces.
