## Summary

Domestique is a hands-on Revenue Operations consultancy partnering with high-growth companies ($5M-$100M ARR) to reduce operational friction across Sales, Marketing, and CS. This Senior GTM Engineer / AI GTM Architect role is client-facing — architect and implement AI-forward GTM systems from tech stack rationalization to AI-powered automation and intelligent workflows. Remote-first, hourly model ($85-$100/hr). Denver-based.

## Responsibilities

- Design GTM systems architecture — translate revenue strategy into end-to-end tech stack roadmaps, evaluate systems, rationalize tooling
- Own implementation from roadmap to reality — configure systems, build automations, integrate tools, deploy AI solutions
- Architect AI-enabled workflows for prospecting, enrichment, lead routing, forecasting, attribution, reporting, and enablement
- Build automation across the revenue stack using Clay, Make, n8n, Zapier, APIs, and webhooks
- Deploy AI practically using Claude, ChatGPT, Gemini for segmentation, messaging, scoring, research, forecasting, and decision support
- Operationalize Domestique best practices — embed RevOps frameworks into clean, scalable system design
- Ensure data integrity and governance — balance speed with structure, protect data quality while enabling experimentation
- Produce clear documentation and system maps for team adoption

## Requirements

### Must-Have

- 7+ years in Revenue Operations, GTM Operations, Growth Engineering, Marketing Operations, or similar
- AI fluency — hands-on experience implementing Clay, Make, n8n, Zapier, Claude, ChatGPT, Gemini in production
- Technical depth — APIs, webhooks, automation logic, data mapping, system integrations, prompt design
- CRM expertise — strong HubSpot and/or Salesforce experience
- Systems mindset — design scalable architecture across the full GTM tech stack
- Analytical rigor — tie implementations back to revenue outcomes
- Clear communication to both technical and non-technical stakeholders

### Nice-to-Have

- Building AI-driven outbound or enrichment engines
- Modern data infrastructure (Snowflake, BigQuery, reverse ETL tools)
- Supporting high-growth SaaS or B2B organizations
- Fractional or agency environment experience
- Prompt engineering frameworks and LLM evaluation methods

## Key Concepts

- **Tech Stack Rationalization**: Auditing a company's existing GTM tools to identify redundancies, gaps, and overlapping functionality — then designing a streamlined architecture that reduces cost and complexity while improving data flow. A core deliverable for this consultancy role.
- **Reverse ETL**: The process of pushing transformed data from a data warehouse (Snowflake, BigQuery) back into operational tools like CRMs and marketing platforms — enabling GTM teams to use enriched, warehouse-quality data in their daily workflows.
