## Summary
Ceros is a no-code platform for interactive digital experiences (customers include Workday, Colliers, McKinsey — 80% longer engagement, 20%+ higher conversion). The Senior GTM Engineer joins Revenue Operations to build AI-powered workflows across GTM systems. Sits at the intersection of RevOps, engineering, and AI automation.

## Responsibilities
- Design, implement, and maintain workflows across GTM systems (CRM, marketing automation, CS, billing)
- Build scalable automations using Workato, Zapier, Make, n8n, or native APIs
- Optimize routing, enrichment, signaling, and lifecycle logic
- Create AI-powered automations, copilots, and agents for Sales, CS, and Marketing
- Build LLM-based tools using APIs, embeddings, and orchestration frameworks
- Implement advanced CRM logic (custom objects, validation, scoring, triggers, flows)
- Build and manage data pipelines across GTM systems and warehouse
- Translate business needs into technical solutions

## Requirements

### Must-Have
- 3+ years in RevOps engineering, systems engineering, or sales/marketing ops engineering
- Strong CRM experience (HubSpot preferred)
- Automation/integration platforms (Workato, Zapier, n8n, tray.io, Make)
- Solid API skills (REST APIs, JSON parsing, webhook logic)
- Scalable workflow architecture design
- GTM process understanding (lead lifecycle, pipeline stages, opportunity management)
- SQL or basic scripting

### Nice-to-Have
- Data engineering or ETL tools
- Event-driven architectures
- Sales/CS playbooks and GTM motions
- Product-led or usage-based business models
- SaaS or B2B tech experience

## Key Concepts
- **AI copilots and agents for GTM**: Building LLM-powered tools that assist Sales, CS, and Marketing teams — using embeddings, orchestration, and APIs. Emerging capability.
- **Technical CRM ownership**: Going beyond basic HubSpot setup — custom objects, validation rules, scoring models, triggers, flows. The engineering side of CRM.
