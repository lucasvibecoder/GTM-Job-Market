## Summary

Quzara is a cybersecurity firm focused on government contracting. This GTM Engineer role is unusually technical — you deploy React-based web assets, architect data scraping pipelines for government spend analytics, and engineer AI-driven workflows within HubSpot using OpenAI and Claude APIs. Also owns Lemlist campaign infrastructure, Zapier automation, SaaS governance/security hygiene, and SEO monitoring via Semrush. **A true full-stack builder role** that bridges code and commerce, requiring comfort in a terminal, a React codebase, and SaaS admin settings equally.

## Responsibilities

- Deploy and maintain high-performance web assets built in React; ensure technical SEO indicators are met
- Own campaign engineering: configure and manage Lemlist sequences, infrastructure health, domain reputation, and lead-capture integration
- Engineer automated scrapers to extract intelligence from web sources and government spend databases; transform raw data into dashboards for sales targeting
- Integrate OpenAI and Claude into HubSpot to automate lead intelligence, content personalization, and behavioral triggers; use Claude Code to prototype internal GTM tools
- Develop and maintain complex Zapier workflows ensuring data synchronization between web assets and GTM tools
- Manage GTM SaaS subscription lifecycle: user provisioning, license transfers, password validation, SSO/security standards compliance
- Monitor SEO health and competitor indicators using Semrush; translate findings into GTM strategy adjustments

## Requirements

### Must-Have
- 3+ years in GTM Engineering, Growth Engineering, or Full-Stack Marketing Operations
- React.js, Git, and modern web hosting/deployment workflows
- Advanced HubSpot configuration
- Lemlist infrastructure and campaign engineering
- Zapier automation
- Proficiency with Claude Code, OpenAI APIs, and terminal-based scripting for data extraction
- Advanced SEO/SEM analytics (Semrush) and data visualization for government spend analysis
- Strong background in data scraping and managing large, complex datasets
- Experience with Identity and Access Management (IAM) for SaaS environments
- U.S. citizenship required

### Nice-to-Have
- HubSpot Architect/Admin Certifications
- Zapier Expert status
- AWS/Cloud deployment certifications

## Key Concepts

- **Government Spend Analytics**: Mining public government procurement databases (e.g., USAspending.gov, FPDS) to identify which agencies are buying what, and using that intelligence to target outbound to the right government buyers. Unique to gov-tech/cybersecurity GTM.
- **Domain Reputation Management**: Maintaining the health and deliverability of email-sending domains used in outbound campaigns — monitoring blacklists, warming domains, rotating infrastructure to keep inbox placement rates high.
- **SaaS Governance**: Managing the lifecycle of all GTM SaaS tools (provisioning, deprovisioning, license management, SSO enforcement) — more common in security-conscious companies where every API key and login is treated with rigor.
