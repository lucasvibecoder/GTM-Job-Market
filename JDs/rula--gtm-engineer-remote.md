## Summary

Rula is a remote-first mental healthcare platform dedicated to making mental health accessible and destigmatized. This GTM Engineer is building the company's first GTM Engineering function — using agentic coding tools to design and ship the automation layer for sales and customer success. Heavy emphasis on AI agents (RAG, context engineering), Clay workflows, and code-driven solutions. Ground-floor role working directly with sales leaders.

## Responsibilities

- Design and ship the automation layer for sales and customer success teams
- Build workflows that save 50+ hours/week of manual effort
- Build AI agents with measurable efficiency gains
- Create code-driven solutions that turn manual processes into scalable systems
- Transform raw data into actionable intelligence
- Rapidly prototype solutions and prove measurable impact working directly with sales leaders

## Requirements

### Must-Have

- 5+ years in GTM-adjacent roles (Revenue Operations, Sales Operations, Marketing Operations, Solutions Engineering, GTM tooling)
- Agentic engineering fluency — daily use of coding agents like Claude Code and Cursor to write, test, and iterate code
- Hands-on Clay experience and GTM automation tools (n8n, trigger.dev, pipedream) building bespoke workflows
- Deep CRM understanding (object models, data relationships, lifecycle stages, GTM data flows)
- API and integration fundamentals (REST APIs, webhooks, authentication, system connectivity)
- Demonstrated building portfolio (GitHub, shipped bespoke solutions, technical work samples)
- AI/LLM building experience (RAG systems, context engineering, deployed agents built with code)

### Nice-to-Have

- SQL proficiency (complex queries, CTEs, window functions for GTM data analysis)
- Python scripting and data manipulation
- Data warehouse experience (Snowflake, BigQuery, dbt)
- Attribution models and GTM crediting
- Healthcare industry or highly regulated environments
- Startup/high-growth agile environment experience
- Salesforce expertise (Apex, AgentForce, Einstein)

## Key Concepts

- **Agentic Engineering**: Using AI coding agents (Claude Code, Cursor) as primary development tools — not just for autocomplete, but for full write-test-iterate cycles to build production systems faster.
- **Context Engineering**: The practice of designing and managing the information context fed to LLMs to improve agent accuracy and reliability — includes prompt structure, retrieval-augmented generation, and memory management.
