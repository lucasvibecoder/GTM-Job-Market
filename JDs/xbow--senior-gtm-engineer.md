## Summary

XBOW builds the world's first autonomous pentester, powered by AI. Backed by Sequoia Capital and Altimeter, the team includes creators of GitHub Copilot and GitHub Advanced Security. This Senior GTM Engineer owns the technical architecture of the GTM engine — with a primary focus on driving Product-Led Growth by turning product usage data into real-time sales signals. Heavy Salesforce engineering, multi-layer enrichment orchestration (ZoomInfo + HG Insights + MadKudu), and treating the GTM stack with engineering rigor via GitHub.

## Responsibilities

- Architect the "Product-to-Sales" data pipeline, ensuring product signals are cleaned, routed, and surfaced to GTM teams in real-time
- Manage a multi-layered enrichment strategy across ZoomInfo (firmographics), HG Insights (technographics), and MadKudu (predictive scoring)
- Design multi-step automations using Zapier while evaluating enterprise-grade middleware (Clay, Tray.io, Make)
- Perform advanced Salesforce engineering and maintain a "Source of Truth" in Notion for all technical schemas and GTM logic
- Act as the technical bridge between GTM Ops (Sales, Marketing, CS) and Product Leadership
- Ensure GTM stack is treated with the same rigor, version control, and documentation standards as core product via GitHub

## Requirements

### Must-Have

- Deep Salesforce architecture/API experience and advanced Notion database management
- Expert-level Zapier knowledge; experience with Clay, Tray.io, or Make highly preferred
- Comfortable operating in GitHub for version control and documentation
- Proven track record managing ZoomInfo alongside HG Insights

### Nice-to-Have

- SQL proficiency for complex data warehouse queries (Snowflake, BigQuery, Redshift)
- Python scripting for data manipulation, custom API scripts, or Lambda functions
- Reverse ETL experience with Census or Hightouch
- REST API and webhook mastery for non-native integrations
- Data modeling and schema design for GTM data lake scalability
- GDPR/CCPA compliance and SOC2 familiarity

## Key Concepts

- **Product-Led Growth (PLG) Engineering**: Building the data pipeline that converts product usage signals into actionable sales intelligence — the technical layer that makes self-serve product activity visible and prioritized for sales teams.
- **Predictive Scoring Orchestration**: Using tools like MadKudu to combine firmographic (ZoomInfo) and technographic (HG Insights) data points into a unified scoring model that prioritizes accounts by likelihood to convert.
- **Reverse ETL**: Syncing transformed data from a warehouse (Snowflake, BigQuery) back into operational tools like Salesforce — the opposite direction of traditional ETL, using tools like Census or Hightouch.
