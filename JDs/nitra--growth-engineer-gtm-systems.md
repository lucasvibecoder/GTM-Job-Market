## Summary
Nitra builds fintech and software solutions for healthcare practices, backed by Andreessen Horowitz and NEA with unicorn-track trajectory. The Growth Engineer (GTM Systems) is a heavily technical role requiring a CS degree and SFDC certification — owns data pipelines (Fivetran, BigQuery, Sigma), Salesforce administration, Clay automation, and web properties on Replit. Hybrid in NYC (4 days in-office).

## Responsibilities
- Build and maintain robust data pipelines using Fivetran, BigQuery, and Sigma for a single source of truth
- Architect deep lead enrichment workflows and automated outbound campaigns for high-volume pipeline
- Manage the full marketing tech stack including CRM integrations, drip campaigns, cookie tracking, and intent signal ingestion
- Own Salesforce administration including complex API integrations and data integrity
- Set up and maintain web properties on Replit and leverage Clay for advanced sales automation
- Deploy AI tools to automate repetitive marketing tasks and enhance SEO campaign efficiency
- Collaborate cross-functionally to translate business requirements into scalable technical infrastructure

## Requirements

### Must-Have
- Salesforce (SFDC) Certification
- Bachelor's degree in Computer Science, Software Engineering, or related Engineering field
- Advanced Python (data manipulation, automation scripts)
- Advanced TypeScript/JavaScript (web development, server-side logic)
- Advanced SQL (complex data modeling, warehouse management)
- Deep expertise in data warehouse management and pipeline tools (Fivetran, BigQuery, Sigma)
- Proven experience managing APIs and complex CRM integrations
- Expertise in Clay and drip campaign software
- Strong understanding of modern marketing tech stack (SEO tools, cookie tracking, intent data)
- Hands-on experience with Replit for web development and deployment
- Track record using AI tools for GTM operations

### Nice-to-Have
- None explicitly listed

## Key Concepts
- **Data Control Plane for GTM**: Using Fivetran (ingestion) + BigQuery (warehouse) + Sigma (visualization) as a unified pipeline that creates a single source of truth for all GTM data — enabling reliable reporting and automation.
- **Intent Signal Ingestion**: Capturing buying signals from third-party providers (cookie tracking, intent data) and feeding them into CRM workflows to prioritize outbound efforts.
