## Summary

Hiya builds AI voice technology that analyzes 28B+ calls/month to protect 550M+ users from spam and scams, partnering with carriers like AT&T and Samsung. This Sr. GTM Engineer role owns the company website as a revenue-driving growth engine — leading a migration from Webflow to a headless CMS (React/Next.js), integrating AI agents for personalization and content generation, and expanding internationally across 20+ markets. Note: LinkedIn lists this as "Sr. GTM Engineer, Marketing" but the JD body titles it "Senior Marketing Engineer – Web & AI" — both titles refer to the same role.

## Responsibilities

- Own the end-to-end buyer journey across awareness, consideration, and conversion stages
- Lead migration from Webflow to a modern headless CMS, architecting a scalable React/Next.js frontend
- Build modular systems for rapid campaign and landing page deployment across enterprise and PLG segments
- Integrate AI agents and LLM APIs for dynamic content generation, personalized experiences, SEO/GEO optimization, and scalable localization
- Own international website expansion across 20+ markets with multi-language, multi-region architecture
- Architect and implement event tracking, data layers, and attribution frameworks using GA4, Amplitude, and Google Tag Manager
- Ensure clean data flow between website, CRM, marketing automation, and PLG systems
- Develop hypotheses and testing roadmaps to improve top page performance, owning website KPIs
- Maintain best-in-class Core Web Vitals, performance benchmarks, UX clarity, and accessibility standards

## Requirements

### Must-Have
- 5–8+ years in growth engineering, frontend engineering, or product-focused web development (B2B SaaS/PLG preferred)
- Strong React/Next.js experience building scalable, high-performance web applications
- Hands-on experience leading headless CMS implementations or migrations
- Proficiency with GA4, Amplitude, and A/B testing / experimentation frameworks
- Strong understanding of technical SEO, GEO, AEO, and performance optimization
- Experience integrating APIs and third-party systems into web platforms
- Ability to translate marketing strategy into technical execution

### Nice-to-Have
- Familiarity with CRM and marketing automation systems (HubSpot, Salesforce, Segment)

- Strong cross-functional collaboration skills across marketing, product, and engineering

## Key Concepts

- **Headless CMS**: A content management architecture that decouples the content backend from the frontend presentation layer, enabling developers to build custom frontends (e.g., React/Next.js) while content teams manage content independently.
- **GEO (Generative Engine Optimization)**: Optimizing web content to appear in AI-generated search results and summaries, complementing traditional SEO by targeting how LLMs surface and cite content.
- **AEO (Answer Engine Optimization)**: Structuring content to directly answer user queries in featured snippets and voice search results, focusing on question-based search intent.
- **Core Web Vitals**: Google's set of performance metrics (LCP, FID, CLS) measuring real-world user experience for loading, interactivity, and visual stability.
