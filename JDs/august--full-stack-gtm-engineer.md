## Summary

**August** (august.law) builds AI to help midsized law firms draft, review, and analyze documents at scale — backed by NEA, Pear VC, and Stanford Law. Claiming >4x revenue growth in four months. This **Full-Stack GTM Engineer** builds the entire technical infrastructure for customer acquisition: outbound pipelines (Clay, Amplemarket, Apollo), LinkedIn automation, SEO systems, analytics infrastructure (PostHog, Statsig), referral platforms, and Google Ads automation. Unusually engineering-heavy for a GTME role — requires SDE2/SDE3 experience at tier-one tech companies and a top engineering institution degree.

## Responsibilities

- Implement prospecting pipelines using Clay, Amplemarket, and Apollo APIs with A/B testing for email sequences and messaging variants
- Build lead enrichment and scoring with automated quality checks
- Optimize deliverability through SPF/DKIM/DMARC configuration and domain warmup
- Design LinkedIn prospecting automation with rate-limiting and anti-detection systems
- Automate technical SEO tasks including schema markup, sitemap updates, and content optimization pipelines
- Integrate Google Search Console API for ranking tracking
- Instrument event tracking across acquisition channels using PostHog
- Build data warehouse ETL aggregating multiple tool sources
- Design A/B testing framework with Statsig
- Develop real-time dashboards and attribution modeling
- Build referral platform with tracking and viral loop mechanics
- Integrate Google Ads API for programmatic campaign management
- Build HubSpot API integrations and sales operations automation

## Key Concepts

- **SPF/DKIM/DMARC**: Email authentication protocols that verify sender identity and prevent spoofing — critical for outbound email deliverability. Misconfiguration tanks open rates and lands messages in spam.
- **Domain Warmup**: Gradually increasing email send volume on a new domain to build sender reputation with email providers — rushing this triggers spam filters and kills deliverability for the entire outbound motion.
- **Statsig**: An experimentation platform for running feature flags and A/B tests at scale — provides statistical rigor for measuring which variants win across product and growth experiments.
