# Growth / Technical GTM Engineer

Writes code to build internal tools, run experiments, and automate GTM workflows. Full-stack capable. Thinks in hypotheses and metrics. Builds the infrastructure that GTM runs on.

## At a Glance
- **Typical tools**: Python, SQL, JavaScript, n8n/Make/Zapier, APIs/webhooks, data warehouse, A/B testing frameworks
- **Typical titles**: GTM Engineer, Growth Engineer, Founding GTM Engineer, Data Engineer - GTM
- **YOE**: 2-5
- **Salary range**: $120,000 - $225,750 base (median $167,500)
- **Key differentiator**: Writes production code. Experiment-driven. Measures everything.
